What is Cloud Computing?

Cloud computing is the on-demand delivery of IT resources over the Internet with pay-as-you-go pricing. Instead of owning and maintaining physical data centers and servers, you access technology services — compute, storage, databases — on an as-needed basis from a cloud provider like AWS.

The NIST definition (SP 800-145) forms the foundation of many exam questions. Know all 5 characteristics.
NIST SP 800-145 DefinitionAWS: What is Cloud Computing

5 Essential Characteristics (NIST)

Shared Responsibility Model

⚠️ The #1 most-tested concept on the CCP exam. AWS = Security OF the Cloud. You = Security IN the Cloud.
Official Shared Responsibility Model
ServiceAWS ManagesCustomer Manages
Always AWSPhysical data centers, hardware, hypervisor, network infra, AZ power/cooling—
EC2 (IaaS)Hardware, hypervisor, host networkingGuest OS patching, apps, security groups, data, IAM
RDS (PaaS)Hardware, OS, DB engine patches, automated backupsDB users/passwords, encryption choice, security groups
LambdaOS, runtime, scaling, execution environmentFunction code, IAM execution role, VPC config
S3Infrastructure, 11-nines durabilityBucket policies, ACLs, encryption, public access settings
DynamoDBEverything except data and accessTable access policies, data, encryption choice
Rule of thumb: The more managed the service, the more AWS takes on. EC2 → you patch OS. RDS → AWS patches DB engine. Lambda → AWS manages everything except your code.

Database Migration

  • AWS DMS — Database Migration Service. Homogeneous (MySQL→MySQL) or heterogeneous (Oracle→Aurora). Continuous replication for minimal downtime.
  • AWS SCT — Schema Conversion Tool. REQUIRED for heterogeneous migrations. Converts source schema to target DB engine syntax. Use with DMS for cross-engine migrations.

AWS Pricing Models

AWS Pricing WhitepaperFree Tier DetailsPricing Calculator

3 Fundamental Pricing Principles

  • Pay as you go — No upfront, no contracts, no termination fees. Pay only for what you consume.
  • Save when you reserve — Commit 1 or 3 years → up to 72% discount. Options: All Upfront (max discount), Partial Upfront, No Upfront.
  • Pay less using more — Volume tiers: the more you use S3, CloudFront, data transfer — the lower the per-unit cost.

Data Transfer Pricing — CRITICAL EXAM TOPIC
